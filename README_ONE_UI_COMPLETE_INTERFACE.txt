ITL351 Complete Network Simulator - One Unified Web Interface

Open:
web_interface/index.html

This updated interface includes:
1. One UI containing all three sections.
2. Quick scenarios for topologies, error control, access control, dynamic routing, ARP, routing table, MAC table, transport and application services.
3. Manual topology builder: add Host, Server, Hub, Switch, Router.
4. Auto IP assignment and manual IP/CIDR entry.
5. Complete layer details table for Physical, Data Link, Network, Transport and Application layers.
6. Individual per-packet work table showing what every layer does when a packet is sent.
7. Live MAC/CAM table, ARP table, routing table, error/access control table and dynamic protocols table.
8. Packet animation across the visual topology.

Python simulator is still included in network_simulator/ and can be run with:
cd network_simulator
python main.py
