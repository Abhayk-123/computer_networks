ITL351 Computer Networks Lab - One UI Simulator
================================================

Open this file in browser:

    web_interface/index.html

This is one single web interface containing all three submissions:

1. Submission 1 UI
   - Physical layer base simulator
   - Hub broadcast, star topology
   - Switch MAC learning, framing, CRC, CSMA/CD, ARQ comparison

2. Submission 2 UI
   - Network layer simulator
   - CIDR/VLSM IPv4 addressing
   - ARP request/reply
   - Router, routing table, static routing
   - RIP distance-vector convergence
   - TTL decrement/drop and longest prefix matching

3. Submission 3 UI
   - Transport + Application layer simulator
   - Port numbers, sockets, multiplexing
   - TCP 3-way handshake and sliding window
   - UDP datagram
   - HTTP, FTP and DNS services

4. Full Stack Demo
   - HostA HTTP client to HostB HTTP server through Router1
   - APP -> TCP -> NET -> DL -> PHY encapsulation and reverse decapsulation

No separate web pages are required. All sections run inside the same UI.
